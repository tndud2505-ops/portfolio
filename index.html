동아전기부품 · 모터 제어

LS Automotive · 바디 제어기

MOBASE ASEC · AUTOSAR 자동화

차량 소프트웨어 설정 자동화 및 검증 도구 개발

Discord Remote Automation

Discord에 이슈를 남기면 AI가 검토하고 작업 결과를 되돌려주는 원격 자동화

로컬 PC 앞에 있지 않아도 Discord에 요청을 남기면 AI가 내용을 검토하고, 필요한 작업을 수행한 뒤 진행 상황과 결과를 다시 메시지로 업데이트하는 구조입니다. 단순 알림봇이 아니라 이슈 등록, 주기적 검토, 작업 실행, 상태 보고가 이어지는 원격 작업 흐름으로 만들었습니다.

  • Discord 이슈 기반 작업 요청
  • AI 정기 검토와 답변 생성
  • 작업 실행 후 진행 상태 업데이트
  • 로그 확인과 원격 중지/재시작
해결한 문제
외부에 있을 때는 로컬 터미널을 계속 볼 수 없어 작업이 멈췄는지, 답변이 필요한지, 완료됐는지 늦게 알게 됩니다.
구현한 방식
Discord 이슈 입력, 주기적 AI 검토, 작업 실행, 진행 로그, 완료/보류 상태 업데이트가 하나의 흐름으로 연결되도록 구성했습니다.
확인 가능한 결과
사용자는 Discord에서 요청을 남기고, AI의 검토 결과와 작업 진행 상황을 메시지로 확인할 수 있습니다.

Multi-Agent Harness Engineering

여러 Codex 에이전트를 나누어 실행하고 조율하는 멀티 에이전트 하네스

큰 작업을 하나의 AI 세션에 모두 맡기는 대신, 여러 Codex 에이전트가 각자 작업 환경을 갖고 조사, 구현, 검증, 리뷰를 병렬로 진행하도록 만든 작업 하네스입니다. 에이전트별 결과를 메인 세션에서 공유·조율하고, 충돌이나 누락을 리뷰 루프로 다시 잡는 구조를 설계했습니다.

  • 역할별 Codex 세션 분리
  • tmux형 멀티 패널 작업 환경
  • 에이전트 간 결과 공유와 조율
  • 리뷰 루프 기반 통합 검증
해결한 문제
큰 작업을 단일 에이전트로 처리하면 조사, 구현, 리뷰가 섞여 누락이나 판단 오류를 늦게 발견하기 쉽습니다.
구현한 방식
역할별 Codex 에이전트가 각자 작업 환경을 갖고 실행하며, 중간 결과를 공유하고 메인 세션에서 조율·통합하는 하네스 구조로 설계했습니다.
확인 가능한 결과
분석, 구현, 검증, 리뷰를 병렬로 확보하고 에이전트 간 피드백을 남겨 충돌·누락·품질 문제를 더 빨리 발견할 수 있습니다.

Used Market Tracking System

여러 중고거래 사이트의 매물을 수집하고 비교하는 통합 추적 시스템

중고나라, 번개장터 등 여러 사이트의 관심 키워드 매물을 모아 가격, 상태, 등록 시점, 확인 포인트를 비교하는 도구입니다. 반복 검색을 줄이고, 조건에 맞는 후보를 빠르게 추려볼 수 있도록 매물 정보를 한 화면에서 정리했습니다.

  • 키워드 기반 매물 수집
  • 가격 범위와 후보 비교
  • 중복 후보와 확인 포인트 정리
  • 관심 매물 빠른 발견
해결한 문제
사이트별로 검색 조건과 가격 기준이 흩어져 있어 좋은 매물을 놓치기 쉽습니다.
구현한 방식
키워드, 가격 범위, 후보 목록, 확인 포인트를 한 화면에 정리했습니다.
확인 가능한 결과
반복 검색 시간을 줄이고, 후보별 가격·상태 차이를 한 화면에서 비교할 수 있습니다.

Stock Analysis & Auto-Trading Workbench

차트 조건 분석과 자동매매 실행 단계를 분리한 주식 워크벤치

종목 후보, 열 기반 조건, 차트 이론, 포지션 상태, 자동매매 실행 전 체크 항목을 분리해 보여주는 주식 자동매매 워크벤치입니다. 자동매매를 바로 실행하는 화면이 아니라, 조건 검출과 실행 경계를 분리해 사람이 매매 근거를 확인한 뒤 자동매매 상태를 관리할 수 있도록 구성했습니다.

  • 열 기반 조건과 차트 패턴 검토
  • 후보 종목·포지션·주문 이력 분리
  • 자동매매 실행 전 안전 체크
  • 실거래/모의 실행 상태 기록
해결한 문제
조건 분석, 차트 해석, 매매 후보, 자동매매 실행 상태가 섞이면 어떤 근거로 실행됐는지 추적하기 어렵습니다.
구현한 방식
열 기반 조건과 차트 이론을 후보 필터에 반영하고, 포지션·주문 이력·자동매매 상태를 분리해 판단과 실행 흐름을 보이게 했습니다.
확인 가능한 결과
조건 충족 근거, 보유 포지션, 실행 상태를 함께 확인할 수 있어 자동매매 전 검토와 실행 후 기록 추적이 쉬워집니다.

Web-based MCU Simulator

코드를 입력하고 전압·전류·레지스터 변화를 확인하는 웹 기반 MCU 교육 시뮬레이터

브라우저에서 직접 코드를 입력하고 실행하면서 가상 보드, 레지스터 값, 전압·전류 상태, 신호 흐름을 함께 확인하는 MCU 교육용 사이트입니다. 실제 장비 없이도 코드와 레지스터 설정이 회로 상태에 어떤 영향을 주는지 가상화해 반복 학습할 수 있게 만들었습니다.

  • 브라우저 코드 입력과 가상 실행
  • 레지스터 설정값 실시간 확인
  • 전압·전류·신호 흐름 시각화
  • 교육용 MCU 동작 가상화
해결한 문제
실제 코드를 입력해도 레지스터, 전압·전류, 보드 상태 변화를 따로 확인해야 하면 MCU 실행 흐름을 이해하기 어렵습니다.
구현한 방식
코드 입력, 빌드·실행 제어, 레지스터 설정, 가상 보드의 전압·전류와 신호 흐름을 브라우저 화면에 묶었습니다.
확인 가능한 결과
별도 장비 없이도 코드 실행에 따른 레지스터와 신호 상태 변화를 웹 화면에서 반복 학습할 수 있습니다.
