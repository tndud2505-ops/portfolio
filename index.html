센서형 히트펌프 액추에이터 / 센서리스 PT 액추에이터 제어

PMSM 모터를 기어를 통해 액추에이터로 전달하고, 냉매 유로가 목표 위치까지 열리고 닫히도록 제어했습니다. 상전환, zero crossing 감지, sensorless 제어, STALL 감지, 재시도, 목표 위치 정지처럼 실제 동작 안정성에 영향을 주는 판단 로직을 직접 구현했습니다.

LS Automotive · 바디 제어기

SCM / PSU / 통합제어기 선행

조향 컬럼 모듈, 파워 시트 유닛, 통합 제어기 선행 과제를 통해 차량 기능이 사용자 입력, 통신, 진단, 기능안전 문서와 어떻게 연결되는지 경험했습니다. 구현뿐 아니라 문서 정리와 협업 산출물 관리까지 함께 다뤘습니다.

MOBASE ASEC · AUTOSAR 자동화

ARXML 기반 코드 생성 및 검증 자동화

AUTOSAR ARXML을 읽고 C 코드 생성과 빌드 검증까지 이어지는 자동화 흐름을 만들었습니다. 반복 설정을 줄이고, 생성 결과를 WSL/GCC 환경에서 확인할 수 있게 만드는 방향으로 작업했습니다.
